# Accessibility (RGAA 4.1.2-inspired)

Goal: improve accessibility on critical user journeys using RGAA 4.1.2 (AA-level) as inspiration, without `.gouv.fr` compliance formalism.

## Scope

- `Auth`
- `Hub`
- `Admin`

## Target

- `RGAA 4.1.2 AA` inspired target
- Pragmatic approach: quick wins + targeted audit + iterative fixes

## Contents

- `rgaa-4.1.2-audit-checklist.md`: targeted audit checklist
- `rgaa-4.1.2-report-template.md`: report template (`OK`/`KO`/`NA`)
- `rgaa-4.1.2-backlog-template.md`: prioritized backlog (`P0`/`P1`/`P2`)
- `accessibility-declaration-template.md`: optional accessibility declaration page

## Minimum Validation

- Automated checks: `Lighthouse` + `axe` on critical pages
- Manual checks: keyboard only, screen reader (`NVDA` / `VoiceOver`), zoom `200%` to `400%`

## Scope Decisions

- No multi-year accessibility scheme
- No annual action plan
