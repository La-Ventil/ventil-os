# Product Documentation

This section documents product-facing source material used as references.

## Source Files

Product source files live in `docs/contributor/product/`:

- `docs/contributor/product/Ventil O.S. - liste des fonctionnalités - RAW DATA.csv`
- `docs/contributor/product/screens/`

## File Stability Rule

Do not rename designer-provided product files.

This applies to:

- screenshots
- exported CSV files
- other designer-exported product reference files

Reason:

- these files are re-exported and re-imported regularly
- keeping the original filenames makes replacement mechanical and low-risk
- changing names adds avoidable maintenance noise

We may reorganize surrounding documentation, but product source filenames should remain unchanged.
